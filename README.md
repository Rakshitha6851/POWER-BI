# POWER-BI
Smart Farming Crop Yield Insights Dashboard
Project Overview

This project presents a Smart Farming Crop Yield Insights Dashboard built using Microsoft Power BI Desktop.
The dashboard analyzes agricultural data to understand crop yield performance, environmental factors, and regional differences affecting crop productivity.

The goal of this project is to provide data-driven insights for better farming decisions by analyzing factors such as rainfall, temperature, soil moisture, and crop types.

Dataset

The dataset contains agricultural information including:

Crop type

Region

Farm ID

Rainfall (mm)

Temperature

Soil moisture

NDVI index

Irrigation type

Crop disease level

Yield per hectare

Latitude and Longitude

This data helps analyze crop productivity and environmental conditions affecting farming.

Tools Used

Microsoft Power BI Desktop – Data visualization and dashboard creation

Microsoft Excel – Data storage and preprocessing

Dashboard Features

The dashboard includes several visualizations to analyze crop yield patterns.

KPI Cards

Display key metrics such as:

Total Farms

Best Performing Crop

Average Crop Yield

Stacked Column Chart

Compares crop disease severity across different crop types.

Scatter Chart

Shows the relationship between rainfall and crop yield, helping understand how rainfall affects productivity.

Treemap

Displays average temperature and crop yield by crop type.

Column Chart

Shows the number of farms across different regions.

Pie Chart

Shows the distribution of soil moisture across crop types.

Donut Chart

Displays the count of farms in each yield category.

Line Chart

Shows the trend of crop yield across regions and crop types.

Map Visualization

Displays the geographical distribution of crop yield using latitude and longitude.

Dashboard Navigation

The dashboard contains two pages:

Page 1 – Overview

Key metrics

Crop yield analysis

Environmental factor insights

Page 2 – Detailed Analysis

Geographic distribution of farms

Irrigation analysis

Crop performance comparison

Navigation between pages is implemented using Power BI page navigation buttons.

Key Insights

Soybean shows the highest crop yield among the crops analyzed.

Rainfall has a significant impact on crop productivity.

Crop yield varies across different regions.

Environmental factors such as temperature and soil moisture influence crop performance.

Farm distribution varies geographically.
